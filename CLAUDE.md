## Project overview

**Pocket Heist** is a starter project for the Claude Code Masterclass. It is a web app for creating and and managing "tiny missions" with an office mischief theme. Currently a frontend skeleton with no backend, database or authentication implemented.

## Architecture

- **Next.js 16** with App Router
- **React 19** with Typescript 5
- **Tailwind CSS 4** + CSS Modules for styling
- **Vitest** + React Testing Library for unit tests
